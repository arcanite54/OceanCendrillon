## OceanCendrillon

# タイトル
オーシャンサンドリヨン

# ゲーム概要
お城を壊しに来る敵を倒して、お城を60秒間守り切れ！

# 操作説明

Zキー(Enterキー)……決定、弾を撃つ
十字キー(W,A,S,Dキー)……移動
スペースキー……ジャンプ
シフトキー……止まる（方向転換用）
エスケープキー……ゲーム終了
